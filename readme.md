## Sobre o Projeto 
Este **Sistema de cadastro de alunos**  é uma aplicação FullStrack desenvolvida como um projeto de revisão do conteúdo no curso de Desenvolvimento de Sistemas, oferecido pelo SENAI Sorocaba - Gaspar Ricardo Junior, ministrad pelo Prof. [Vedilson Padro](https://github.com/vedilsonpradro). O objetivo do projeto é consolidar conhecimentos em criação de API RestFull, manipulação de Banco de Dados Reçacional e integração com o Frontend usando JavaScript.

A aplicação permite o gerenciamento completo(CRUD) de alunos, incluindo validações de regras de negócio e persistência de dados.

## Tecnologias Utilizadas
### Backend 
- **Java 17:** Principal linguagem
    - **Spring Boot:** Framework para a cricação da API

### Banco de Dados 
- **MySQL:** Banco de Dados Relacional.

### Frontend
- **HTML5 & CSS3:** Estrutura semântica e estilização.
- **JavaScript:** Lógica do cliente e consumo de API.

---   

## Como executar o projeto 

### 1. Banco de Dados 
-  Crie o banco de dados MySQL executando o script SQL: 
```sql
CREATE DATABASE db_cadastro_aluno;
```

### 2. Backend 
- Clonar repositório. 
- Configure o arquivo application.properties com suas credenciais do MySQL.
- A API rodará em: `http://localhost:8080`

### 3. Frontend 
- Navegue até a pasta do Frontend.
- Acesse  o index.html no navegador.
